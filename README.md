# 🎲 智能课堂点名系统
一个简单易用、带图形化界面的Python课堂点名工具，专为老师课堂互动设计。

## ✨ 功能特性
- 📥 Excel一键导入：支持通过xlsx文件批量导入学生姓名、性别、座号信息
- 🎯 多条件筛选：支持按性别、座号单双号、座号范围精准抽取学生
- 🎬 滚动抽奖动画：带名字滚动的点名效果，提升课堂互动趣味性
- 📦 开箱即用：提供打包好的exe文件，小白无需安装Python即可直接使用
- 🔧 一键打包：附带完整配置，可自行修改代码后打包成exe

## 🚀 快速使用（无需安装Python）
1.  点击本页面右侧「Releases」，下载最新版的`roll_call.exe`和`学生信息模板.xlsx`
2.  把两个文件放到**同一个文件夹**里
3.  双击`roll_call.exe`即可运行程序
4.  如需修改学生名单，打开Excel模板编辑，保存后在程序里重新导入即可

## 🔧 开发者运行方法
### 环境要求
- Python 3.7及以上版本

### 安装依赖
打开命令提示符，执行以下命令：
```bash
pip install openpyxl
```

### 运行程序
1.  克隆 / 下载本项目代码
2.  在项目文件夹里打开命令提示符
3.  执行以下命令即可运行：
```bash
python roll_call.py
```
